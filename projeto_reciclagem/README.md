# Plataforma de Doação de Material Reciclável
Um aplicativo web baseado em Flask para conectar doadores e coletores de materiais recicláveis.

## Configuração
1. Instale o Python 
2. Execute `pip install -r requirements.txt`.
3. Execute `python app.py`.
4. Acesse em `http://localhost:5000`.

## Estrutura de Pastas
- `app.py`: Lógica principal da aplicação.
- `templates/`: Modelos HTML.
- `static/`: Arquivos estáticos (CSS/JS).(Opcional)
